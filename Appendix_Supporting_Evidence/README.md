Appendix_Supporting_Evidence files: 

This folder contains appendix supporting evidence used for additional validation during the DDoS attack timeline reconstruction project.

The appendix includes supplementary screenshots, configuration proof, system setup details, command outputs, packet capture verification, and extra forensic observations that support the main project findings.

These materials provide extended documentation for Wireshark analysis, Zeek logs, Loki filtering, Prometheus monitoring, and Grafana dashboard validation beyond the primary practical evidence folder.

The supporting evidence strengthens the final report, SRS poster, and cybersecurity investigation by providing extra proof of system behavior, attack progression, and multi source forensic correlation accuracy.
