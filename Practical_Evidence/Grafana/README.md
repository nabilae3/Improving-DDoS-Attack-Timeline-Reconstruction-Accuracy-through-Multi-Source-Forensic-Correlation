Grafana practical evidence files : 

Grafana was used to monitor system activity and network behavior during the DDoS simulation and file transfer investigation.
The dashboard showed unusual spikes in network traffic, CPU usage, and memory consumption during the same time the suspicious connection between Computer A and Computer B was detected.
The Network Traffic and Network Saturation graphs clearly showed abnormal communication through port 5001, while the CPU and Memory dashboard confirmed increased system activity during the transfer process.
These visual indicators helped the security team identify the attack start time, peak activity, and recovery phase, which improved forensic timeline reconstruction and supported the final investigation findings.
