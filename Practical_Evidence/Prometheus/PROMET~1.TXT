Prometheus was used to collect and monitor real time system performance data during the DDoS simulation and forensic investigation.

It gathered important metrics such as CPU usage, memory consumption, network traffic, disk activity, and system resource utilization from the monitored machine using Node Exporter.

Prometheus was integrated with Grafana to provide continuous monitoring and visualization of these system metrics during the attack and file transfer process.

The monitoring graphs showed abnormal increases in CPU usage, memory load, network traffic, and disk read/write operations during the suspicious communication between Computer A and Computer B.

These spikes helped identify attack initiation, peak activity, and recovery phases more accurately.

Prometheus improved forensic timeline reconstruction by providing measurable system evidence that supported the packet analysis from Wireshark, structured logs from Zeek, and log correlation through Loki.

This strengthened the final investigation by improving accuracy, validation, and confidence in detecting suspicious activity.
