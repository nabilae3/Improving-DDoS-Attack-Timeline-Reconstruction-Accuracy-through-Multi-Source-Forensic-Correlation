Prometheus practical evidence files: 

This folder contains Prometheus monitoring evidence used during the DDoS attack timeline reconstruction project.

Prometheus was used to collect and track system performance metrics such as CPU usage, memory utilization, network activity, and service behavior during the simulated SYN flood attack.

These monitoring results helped detect abnormal resource consumption, validate attack intensity, and confirm the impact of the DDoS attack on system stability.

The Prometheus data supported multi source forensic correlation by providing measurable evidence that matched Wireshark, Zeek, Loki, and Grafana findings.
