Zeek practical evidence files:

This folder contains Zeek network log evidence used during the cybersecurity investigation and DDoS attack timeline reconstruction project.

Zeek was used to generate structured network logs and analyze connection level activity during the simulated SYN flood attack. It helped identify high frequency connections, incomplete TCP handshakes, abnormal session behavior, and suspicious communication patterns between the attacker and victim systems.

The connection logs provided detailed forensic visibility into network behavior, allowing accurate validation of attack initiation, peak attack activity, decline phase, and system recovery.

These Zeek logs strengthened cybersecurity analysis by supporting precise attack timeline reconstruction and improving multi source forensic correlation with Wireshark packet captures, Loki filtering, Prometheus monitoring, and Grafana dashboard evidence.
