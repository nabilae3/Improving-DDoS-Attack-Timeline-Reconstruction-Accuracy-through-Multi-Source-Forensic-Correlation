Wireshark practical evidence files: 

This folder contains Wireshark packet capture evidence used during the cybersecurity investigation and DDoS attack timeline reconstruction project.

Wireshark was used to capture, monitor, and analyze network traffic generated during the simulated SYN flood attack. TCP SYN, ACK, FIN, and RST packets were carefully inspected to identify abnormal communication patterns, suspicious connection attempts, and unauthorized traffic behavior between the attacker and victim systems.

The packet level analysis helped validate attack initiation, peak attack activity, decline phase, and recovery by examining connection frequency, packet retransmissions, incomplete handshakes, and sudden traffic spikes.

These packet captures provided strong forensic evidence for improving cybersecurity analysis, reconstructing the attack timeline accurately, and supporting multi source correlation with Zeek logs, Loki filtering, Prometheus monitoring, and Grafana dashboard analysis.
